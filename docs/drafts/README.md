# ACME Draft Specifications

This directory contains ACME draft specifications not yet published as RFCs.

## Current Drafts

- **[draft-ietf-acme-profiles-00](draft-ietf-acme-profiles-00.txt)** - ACME
  Profiles Extension. Allows servers to offer certificate profile selection
  and clients to indicate preferences. Implemented by Let's Encrypt (Boulder)
  and multiple ACME clients.

## Note

Drafts are subject to change before becoming RFCs. Check the
[IETF Datatracker](https://datatracker.ietf.org/wg/acme/documents/) for
the latest versions.
